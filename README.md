# Video-Editing - Portfolio

A responsive personal portfolio website built with **HTML**, **Tailwind CSS**, and **JavaScript**.
This project **prototype** design in a developer-friendly codebase.
---

## Features
- Responsive Layout (Mobile, Tablet, Desktop)
- TailwindCSS styling via CDN
- Smooth scrolling with introduction
- About, Skills, Services, Work, and Contact Sections
- Sample video embeds for Shorts and Vlogs
- Contact form with dummy JavaScript submission
- Footer with dynamic year and modal dialogs for Terms & Privacy

## Getting Started
1. **Clone or Download** this repository.
2. Place your own images and videos inside the `assets/images` and `assets/video/` folders.
3. Open `index.html` in your browser to view the site.
